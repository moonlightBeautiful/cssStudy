# cssStudy
盒模型
    边框border
        宽度 颜色 样式，也可以三合一，也可以上下左右分别设置。
        默认无边框
    边框之间的距离，外边距 margin
        两个行内元素的左右外边距=左外边距+右外边距。水平相加。
        两个块元素上下外边距=上下块元素最大的上或下外边距
        嵌套元素的外边距=子元素的外边距以父元素的内容为准
    边框和边框内内容的距离，内边距padding
        内边距默认0px
        也可以四个边一起设置，也可以上下左右分别设置。
    标准文档流
        默认的html元素布局方式，排列规则，默认css规则。
    块级元素和行内元素
        块级元素：自动换行。比如div
        行内元素：排满自动换行，比如span
